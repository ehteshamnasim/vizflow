# Screenshots

Place your FlowKit screenshot here as `screenshot.png`

## How to capture:
1. Open https://viz-flow.netlify.app/
2. Take a screenshot (or use the "Image" button to export the canvas)
3. Save as `screenshot.png` in this folder
4. Commit and push to GitHub

The README references this image at:
`https://raw.githubusercontent.com/ehtesham/flowkit/main/assets/screenshot.png`
